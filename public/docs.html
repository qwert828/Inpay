InPay API Docs

Accept payments from MTN, Airtel, Zamtel, SpeedPay and ZedMobile through one simple REST API. All responses are JSON. All amounts are in ZMW.

LIVEhttps://api.inpay.zm/v1
SANDBOXhttps://sandbox.inpay.zm/v1
Get your free sandbox API key at inpay.zm/signup — no approval needed to start testing.

Authentication

Add your API key to every request header:

X-API-Key: inpay_sandbox_xxxxxxxxxxxxxxxx
X-InPay-Timestamp: 1714500000000
Idempotency-Key: unique-uuid-per-request
Content-Type: application/json

Initiate Payment

POST/payments/initiateSend payment request to customer
// Request
{
  "amount":      250.00,
  "currency":    "ZMW",
  "provider":    "MTN",
  "phone":       "+260971234567",
  "description": "Order #1042"
}

// Response
{
  "status":        "processing",
  "transactionId": "uuid-xxxx",
  "reference":     "INP-1714500000-A3B",
  "amount":        250.00,
  "fee":           4.45
}

Check Payment Status

GET/payments/:id/statusPoll transaction status
// Response
{
  "status":    "completed",
  "reference": "INP-1714500000-A3B",
  "amount":    250.00,
  "provider":  "MTN"
}

Generate QR Code

POST/payments/qr/generateCreate static or dynamic QR
// Request
{
  "type":        "dynamic",
  "amount":      250.00,
  "description": "Order #1042"
}

// Response
{
  "qrId":      "uuid-xxxx",
  "qrPayload": "https://pay.inpay.zm/q/uuid",
  "expiresAt": "2026-06-01T12:15:00Z"
}

Supported Providers

MTN       → MTN Mobile Money Zambia  (097/096)
AIRTEL    → Airtel Money Zambia       (095/075)
ZAMTEL    → Zamtel Kwacha             (094)
SPEEDPAY  → SpeedPay / Zanaco         (077)
ZEDMOBILE → ZedWallet (ZedMobile)     (098)
